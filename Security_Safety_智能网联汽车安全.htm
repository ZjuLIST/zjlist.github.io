# Security & Safety: 智能网联汽车安全

汽车行业正处于一个迅猛发展的时期，智能网联与无人驾驶等概念逐渐深入人心，针对智能汽车的安全不仅仅涉及到财产和信息安全，更关键的是其可能存在的人身安全危害。

本次课程首先介绍智能网联汽车技术基础，包括汽车的典型车电网络架构，通信方式等；然后从威胁建模的角度分析智能网联汽车可能存在的安全隐患，包括物理接触式的OBD-II口攻击、近场通信的Wi-Fi\Bluetooth攻击、远程通信的云端\APP安全等；最后对汽车安全研究中几个较为知名的真实攻击案例进行剖析，包括2015年Charlie Miller等人对于Jeep Cherokee的远程攻击和2016年腾讯科恩实验室针对Tesla的远程攻击等。

##  三、真实案例剖析 (60 min)

### 3.1 JEEP Hacking 2015
 
http://icitech.org/wp-content/uploads/2015/09/ICIT-Brief_Whos-Behind-the-Wheel_Car-Hacking2.pdf
 
2015年著名安全研究员Charlie Miller等人对外公布了针对Jeep Cherokee的远程攻击，该案例对汽车行业影响深远。

该研究成果已经对外公布，考虑到整个攻击链包含多个漏洞，本小节对该研究中涉及的漏洞进行逐步剖析，从技术角度还原整个攻击行为。


### 3.2 Tesla Hacking 2016

http://keenlab.tencent.com/en/2016/09/19/Keen-Security-Lab-of-Tencent-Car-Hacking-Research-Remote-Attack-to-Tesla-Cars/

2016年９月腾讯科恩实验室对外公布了针对Tesla Model S的远程攻击，该研究也在汽车行业引起了很大反响。

本小节主要介绍针对特斯拉的攻击链上的多个漏洞，同时对JEEP和Tesla这两个攻击案例进行对比分析，旨在发掘汽车安全研究中的共性。


### 3.3 其它研究案例

简要介绍一些物联网设备的安全攻防案例，旨在分析车联网安全与物联网安全的异同。
